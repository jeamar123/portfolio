lazy.datepicker.js

A customized jquery plugin which allows a user to select a date.

Demo:

<input id="element_id" type="text" placeholder="Select Date" />

Usage:

< ! - - Include necessary scripts and styles - - >

<link rel="stylesheet" type="text/css" href="css/font-awesome.css">

<link rel="stylesheet" type="text/css" href="css/lazy.datepicker.css">

<script type="text/javascript" src="js/jquery.min.js"></script>

<script type="text/javascript" src="js/moment.js"></script>

<script type="text/javascript" src="js/lazy.datepicker.js"></script>

< ! - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - >

$("#element_id" ).lazydatepicker({

parentClass : "lazy-date-picker-initialized",

theme : "default-theme",

date: moment(),

format : 'MMMM DD, YYYY',

weekStartSunday : true,

showHeader : true

});

Options:

Name Type Default Required Description
parentClass String "lazy-time-picker-initialized" Optional Class for the picker's parent element.
theme String "default-theme" Optional Picker color scheme ( "dark-theme", "default-theme" ).
date String/Date moment() Optional Initial Date.
format String 'MMMM DD, YYYY' Optional Format for the inputted date.
weekStartSunday Boolean true Optional If True, Week will start on Sunday.
showHeader Boolean true Optional If True, Datepicker Header will be visible.
